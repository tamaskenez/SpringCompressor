# Experiment log:

I checked the spectrum of amplitude modulating a sine with
- 1 + sine
- sqrt(1 + sine), that is, linear modulation of the power of signal
- db2mag(1 + sine), that is, linear modulation of the dB level of the signal

The first option has the cleanest spectrum, only 2 more peaks at ±df.
The second is worse (2 extra peaks are same, but there's more left and right).
Third is even worse.
This experiment hints at that if we need the cleanest spectrum, the gain-reduction signal must be filtered as linear gain, not as dB.

On the other hand, certain sources suggest that the usual and best sounding location to filter the GR signal is the dB domain.

# Compressor dB transfer curve, soft knee formula

The gradual, even increase of the ratio is exponential R * exp(d). The derivative on transitional region of the knee must increase in an exp(dx) manner.
If the threshold is at x0, the linear region of the curve starts at x1 > x0, the transitional region is x0..x1, then the transitional curve f(x):

f(x0) = x0
f'(x0) = 1
f'(x1) = 1/R

f = A * exp(B * (x - x0)) + C

which results in:

w = x1 - x0;
B = log(1/R) / (x1 - x0);
A = 1 / B;
C = x0 - A;

This results in a most gradual transition from R = 1 to R.

## SSL experiment

Tested 100 Hz sine with SSL G-Bus, 10 ms attack, 100 ms release, ratio = 4
It added odd harmonics, the first being 37 dB lower than the F0.
For 1000 Hz sine, it was 57 dB lower.

The lower the attack the slower the harmonics envelope decay (more higher order harmonics).

Increasing the release lowers the harmonics envelope.
The auto release gives somewhat more harmonics.

FilterBankRmsDetector::FilterBankRmsDetector(double fs, const FilterBankRmsDetectorConfig& config_arg)
    : config(config_arg)
    , fb(
        config.filter_bank.freq_lo_hz / fs * 2,
        config.filter_bank.freq_hi_hz / fs * 2,
        config.filter_bank.filters_per_octave,
        config.filter_bank.Q
      )
    , filter_bank_output_samples(fb.num_filters())
{
    if (auto attack_time_constant = config.lpf.attack_time_constant) {
        lp.spring.reserve(fb.num_filters());
        for (unsigned j = 0; j < fb.num_filters(); ++j) {
            lp.spring.emplace_back(fs);
            const auto fc_hz = fb.center_freq(j) / 2 * fs;
            const auto tau_sec = 1 / (2 * num::pi * fc_hz);
            lp.spring.back().set_critically_damped_with_time_constant(*attack_time_constant, tau_sec * config.lpf.K);
        }
    } else {
        switch (config.lpf.order) {
        case 1:
            lp.first.reserve(fb.num_filters());
            for (unsigned j = 0; j < fb.num_filters(); ++j) {
                const auto fc = fb.center_freq(j);
                lp.first.emplace_back(matlab::butter(1, matlab::FilterType::LowPass{fc / config.lpf.K}));
            }
            break;
        case 2:
            lp.second.reserve(fb.num_filters());
            for (unsigned j = 0; j < fb.num_filters(); ++j) {
                const auto fc = fb.center_freq(j);
                lp.second.emplace_back(matlab::butter(2, matlab::FilterType::LowPass{fc / config.lpf.K}));
            }
            break;
        default:
            assert(false);
        }
    }
}

void FilterBankRmsDetector::process(span<double> samples)
{
    const auto NF = fb.num_filters();
    for (auto& s : samples) {
        fb.process(s, filter_bank_output_samples);
        double sample_power = 0;
        if (lp.first.empty()) {
            assert(lp.first.size() == NF);
            for (unsigned j = 0; j < NF; ++j) {
                sample_power += lp.first[j].process(square(filter_bank_output_samples[j]));
            }
        } else if (!lp.second.empty()) {
            assert(lp.second.size() == NF);
            for (unsigned j = 0; j < NF; ++j) {
                sample_power += lp.second[j].process(square(filter_bank_output_samples[j]));
            }
        } else {
            assert(lp.spring.size() == NF);
            for (unsigned j = 0; j < NF; ++j) {
                sample_power += lp.spring[j].process(square(filter_bank_output_samples[j]));
            }
        }
        s = std::sqrt(sample_power);
    }
}
